# HOW TO INSTALL

1. Copy folders into your Odoo `custom-addons` path:
   - `instalacje24_terrain`
   - `instalacje24_office`
   - `instalacje24_ai_quote` (optional)
2. Restart Odoo service.
3. Go to `Apps` and click `Update Apps List`.
4. Install modules in order:
   1. `Instalacje24 Terrain`
   2. `Instalacje24 Office`
   3. `Instalacje24 AI Quote` (optional)
5. Add your user to group **Instalacje24 User**.
6. Optionally load demo data at DB creation.

Done: office + terrain + optional AI quote are ready.
