# HOW_TO_USE_EMAIL_SIGNATURE

Szablon HTML jest dodany jako `Instalacje24 - podpis email`.

## Gmail
- Ustawienia Gmail > Podpis > wklej HTML wygenerowany z szablonu.

## Outlook
- New Email > Signature > wklej HTML.

## Elementy
- małe logo,
- klikany telefon,
- link WWW,
- linia usług,
- link WhatsApp i Maps.
