# HOW TO CUSTOMIZE PRICING

## Global defaults

1. Open `Settings` → `Sales`.
2. Find app block **Instalacje24**.
3. Set:
   - `Domyślna stawka/h`
   - `Domyślna stawka/km`
   - `Domyślna marża %`
   - `Domyślny VAT %`
   - quick button prices (naprawa/serwis/montaż)
4. Save.

## Estimator formulas

In the same settings block configure:
- koszt materiału / m2,
- koszt łazienki,
- koszt kondygnacji,
- roboczogodziny / m2,
- minimalna marża alert.

## Per-quote override

In `Instalacje24 -> Biuro -> Kalkulator`, each quote is still editable:
- labor hours,
- rates,
- margin,
- VAT,
- material cost,
- travel km.

Totals (`Razem netto`, `Razem brutto`) recalculate automatically.
